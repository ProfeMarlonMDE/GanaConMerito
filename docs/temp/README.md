# docs/temp

> Estado documental: TEMPORAL-NO-CANONICO

Esta carpeta funciona como inbox temporal, staging o parking de material no consolidado.

## Regla de uso
- Nada en `docs/temp/` debe tratarse como fuente de verdad del producto.
- Si un contenido aquí gana vigencia, debe promoverse explícitamente a una carpeta formal de `docs/01-08/` o al directorio temático correspondiente.
- Si solo conserva contexto, debe mantenerse como histórico o descartarse en una fase controlada posterior.

## Regla práctica
No citar `docs/temp/` como canon en decisiones de arquitectura, backlog, delivery, calidad u operación.
